# Nykaa Marketing Campaign Analysis (SQL Project)

## Project Overview
This project analyzes Nykaa's marketing campaign data using SQL to generate business insights and support data-driven decision-making.

## Objectives
- Analyze total revenue generated by campaigns
- Identify top-performing marketing channels
- Evaluate campaign ROI
- Understand customer segment contributions
- Find the best-converting target audiences
- Discover top-performing campaigns

## Tools Used
- MySQL
- SQL
- Jupyter Notebook

## Business Questions
1. What is the total revenue generated?
2. Which marketing channels drive the highest revenue?
3. Which campaign types have the highest ROI?
4. Which customer segments contribute most revenue?
5. Which target audiences convert best?
6. What are the top-performing campaigns?

## Key Skills Demonstrated
- Data Cleaning
- Aggregations
- GROUP BY Analysis
- CASE Statements
- Subqueries
- Business KPI Analysis
- Revenue & ROI Analysis

## Dataset
Nykaa Campaign Marketing Dataset
